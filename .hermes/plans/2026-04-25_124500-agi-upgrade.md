# AGI Upgrade — The Agency

**Goal**: Push The Agency (152 specialized AI agents) toward AGI-level capability by adding:
1. Autonomous tool fabrication (agents create new tools at runtime)
2. Meta-agent spawner (dynamic new agent creation for novel tasks)
3. Self-evolving cron loop (continuous agent improvement)
4. Autonomous exploration (idle-time knowledge discovery)
5. Persistent cross-session agent memory

## Current State
- 152 agent personas (.md files), deepagents SDK v0.4.10
- A2A protocol (Google spec v0.3), MCP tool layer (17 tools)
- Titans surprise-weighted memory, LangSmith-style tracing
- Evolution scheduler (single-agent, on-demand)
- 13 presets, parallel execution groups, voice pipeline, dashboard UI
- Orchestration bridge to sovereign-swarm

## Files to Create/Modify

### New Files
1. `tool_fabricator.py` — Runtime tool synthesis: agents describe new tools in natural language, this module generates and registers LangChain tools
2. `meta_spawner.py` — Watches mission patterns, identifies gaps, creates new agent .md files with dynamic personas
3. `self_evolve_loop.py` — Cron-able script that runs evolution on N weakest agents per cycle, commits improvements
4. `explorer_agent.md` — Autonomous exploration agent persona (new specialist)
5. `explorer_loop.py` — Cron-able script that dispatches the explorer on open-ended research missions

### Modified Files
6. `agency.py` — Add `--explore` and `--evolve-daemon` modes; integrate tool_fabricator into build_subagent
7. `evolution_scheduler.py` — Add batch mode (evolve N agents), add agent scoring/metrics for "weakest" detection
8. `mcp_tools.py` — Add `register_new_tool()` function
9. `memory/titans_memory.py` — Add per-agent memory storage (individual recall beyond shared ledger)
